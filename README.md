# Nivelamento em C# 🕹️🎮🎲
Este repositório é destinado aos meus estudos da **sintaxe** e a as **principais nuances** da
Linguagem de Programação C# com foco em dispôr esse processo de forma organizada através
de **projetos de diferentes níveis de complexidade** e **diferentes conteúdos de programação aplicados no C#**. 

## Objetivos do Repositório 🏹
Os projetos abordam conceitos como:
- Consolidar os fundamentos da linguagem C#;
- Praticar lógica de programação;
- Aprender boas práticas de desenvolvimento;
- Construir uma base sólida para o desenvolvimento com .NET.

## Estrutura do Repositório 🗂️
```
src/
├── CalculadoraSimples/
├── SwitchCase/
├── RepeticoesLoops/
├── ...
```

Cada pasta representa um conceito de programação ou um projeto independente.
